не превращаться в “лубок + ад”

🇷🇺 RussianUI — Design System Specification

(для LLM, строго и формально)

0. Философия стиля

RussianUI — это:

Северная строгость + богатырская основательность + орнаментальный акцент.

Не китч.
Не музей.
Не фольклорный фестиваль.

Это современный цифровой продукт, вдохновлённый:

северным холодом

берестой

старорусской типографикой

тёмным деревом

хохломой (только как акцент)

массивностью форм

1. Визуальная философия
   1.1 Основные принципы

Минимализм с тяжёлым характером.

Контраст выше среднего.

Цвет — акцент, не фон.

Орнамент — точечный, не повсеместный.

UI должен оставаться читабельным и быстрым.

2. Цветовая система (Design Tokens)

Все цвета — через CSS variables.

2.1 Базовые цвета
Light theme
--ru-bg-primary: #f4f1ea; /_ береста _/
--ru-bg-secondary: #e7e1d6; /_ состаренная бумага _/
--ru-surface: #ffffff;

--ru-text-primary: #1c1c1c;
--ru-text-secondary: #4a4a4a;
--ru-text-muted: #7a7a7a;
Dark theme (основная)
--ru-bg-primary: #0f1115; /_ северная ночь _/
--ru-bg-secondary: #1a1d23;
--ru-surface: #20242b;

--ru-text-primary: #f2f2f2;
--ru-text-secondary: #c8c8c8;
--ru-text-muted: #8a8a8a;
2.2 Акцентные цвета

Акцент — это “хохлома”, но дозированная.

--ru-accent-red: #b0232a; /_ насыщенный красный _/
--ru-accent-gold: #c9a227; /_ тёплое золото _/
--ru-accent-ember: #8b1e1e; /_ тёмно-красный _/
--ru-accent-ice: #5b7fa6; /_ холодный синий _/

Правило:

Никогда не использовать одновременно red и gold в большом объёме.

Акцент применяется:

кнопка primary

unread badge

активный элемент

3. Формы и геометрия
   3.1 Скругления

RussianUI — не “айфоновская мягкость”.

--ru-radius-sm: 6px
--ru-radius-md: 10px
--ru-radius-lg: 14px
--ru-radius-xl: 18px

Никаких 28px и “пузырей”.

3.2 Толщина линий

Границы немного плотнее, чем в Material.

border: 1px solid rgba(255,255,255,0.08) (dark)
border: 1px solid rgba(0,0,0,0.08) (light) 4. Типографика

RussianUI = характер через шрифт.

4.1 Основной шрифт

Современный, но с “северным” ощущением:

Inter

Manrope

IBM Plex Sans

4.2 Иерархия
Heading: 600–700
Body: 400–500
Caption: 400

Запрещено:

использовать декоративные кириллические фольклорные шрифты в UI

использовать serif для основного текста

5. Компонентная философия
   5.1 Кнопки

Primary:

фон: accent-red или accent-ice

текст: white

без градиентов

лёгкий внутренний inset-shadow

Secondary:

border + transparent bg

цвет текста = accent

Никакого glass.
Никакого material ripple.

5.2 Chat bubble

Outgoing:

фон: accent-ice (dark mode — чуть затемнённый)

текст: white

Incoming:

фон: surface

border subtle

Скругления:

asymmetrical (как щит)

14px, но один угол чуть острее (10px)

5.3 Badges

Unread badge:

фон: accent-red

текст: white

radius: full (но не oversized)

6. Орнамент (очень важно)

Орнамент используется:

в splash screen

в onboarding

в header background как very subtle pattern

в пустых состояниях

Орнамент:

векторный

монохромный

прозрачность 4–8%

никогда не используется под текстом

7. Анимация

RussianUI не “прыгает”.

Анимации:

быстрые (120–180ms)

ease-out

без bounce

без spring exaggerated

Typing indicator:

пульсация, не прыжки

8. Тени

В dark mode:

box-shadow: 0 2px 8px rgba(0,0,0,0.4);

В light mode:

box-shadow: 0 2px 6px rgba(0,0,0,0.08);

Никаких мягких неоморфических теней.

9. Иконография

Стиль:

простые контурные

1.5px stroke

без rounded ultra style

Можно использовать:

Heroicons

Lucide

10. Тон продукта

RussianUI — это:

уверенный

основательный

спокойный

не глянцевый

не стартапово-яркий

13. RussianUI Design System

Все цвета — только через design tokens.

Никаких inline цветов.

Никаких градиентов.

Никакого glassmorphism.

Никакого material ripple.

Орнамент — только декоративный, не функциональный.

Радиусы не превышают 18px.

Акцент используется дозированно.

Компоненты переиспользуются строго.

Dark theme считается основной.

12. Важно для LLM

LLM обязана:

не изобретать новые цвета

не добавлять random стили

не использовать внешние UI библиотеки со своим стилем

не смешивать Material и RussianUI
